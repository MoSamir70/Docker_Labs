# Problem 1

## Run hello-world

```bash
docker run hello-world
```

## Check container status

```bash
docker ps -a
```

## Start stopped container

```bash
docker start CONTAINER_ID
```

## Remove container

```bash
docker rm CONTAINER_ID
```

## Remove image 
# --> hit the "i" in rmi for image

```bash
docker rmi hello-world
```
